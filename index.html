Команды, которые я уже знаю:

  • Первоначальные настройки:
    • git --version: версия установленного гита
    • git --help: описание команд
    • git config --global user.name: просмотр имени пользователя
    • git config --global user.name "user_name": устанавливает новое имя пользователя
    • git config --global user.email: просмотр почты пользователя
    • git config --global user.email "user@email.ru": устанавливает новую почту пользователя
  • Основные команды:
    • git status: показывает статус текущего локального репозитория
    • git add .: начнет отслеживать все новые файлы или добавит все их изменения в коммит
    • git add file_name: начнет отслеживать новые файл или добавит его изменения в коммит
    • git status path_file: добавляет новый файл под контроль гита (git следит за каждым его изменением)
    • git commit -m "сommit_message": зафиксировать изменения добавленных файлов через git add и дать им комментарий
    • git commit -am "сommit_message": зафиксировать все изменения отслеживаемых файлов и дать им комментарий
  • Работка с ветками:
    • git branch: выводит список веток и указывает в какой находимсмя в данный момент
    • git branch name_branch: создает новую ветку
    • git branch -D name_branch: удаляет указанную ветку
    • git checkout name_branch: переключается на указанную ветку
    • git checkout -b name_branch: создает новую ветку и сразу же переключается на нее
    • git merge name_branch: выполняет слияние активной ветки с указанной
  • Работа с GitHub:
    • OCTOTREE: расширение браузера, для убодной навигации по репозиторию
    • git remote -v: просмотр привязанных удаленных репозиториев
    • git remote add origin https_url: привязка удаленного репозитория к локальному
    • git clone path_repository: клонирование удаленного репозитория (для участия других разработчиков)
    • git push origin master: отправить изменения (коммиты) с локального репозитория на удаленный
    • git pull origin master: забрать изменения (коммиты) с удаленного репозитория на локальный
  • Консольные команды:
    • clear: очищает консоль
    • ls: просмотр текущего каталога
