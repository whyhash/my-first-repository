Команды, которые я уже знаю:

  • git clone path-repository
  • git status: показывает статус текущего локального репозитория
  • git status path-file: добавляет новый файл под контроль гита (git следит за каждым его изменением)
  • git commit -m "Commit-message": зафиксировать изменения добавленных файлов через git add и дать им комментарий
  • git commit -am "Commit-message": зафиксировать все изменения отслеживаемых файлов и дать им комментарий
  • git push: отправить изменения (коммиты) с локального репозитория на удаленный
  • git pull: забрать изменения (коммиты) с удаленного репозитория на локальный
